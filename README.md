# bundle-vanilla-js

바닐라 자바스크립트를 사용하는 새 웹 프로젝트를 빠르게 시작할 수 있습니다

## Usage
- Webpack 5
- Babel
- ESLint
- Prettier
- Axios
- Font-awesome
- Handlebars
- Tailwind

examples 폴더 참고해주세요

## Start
```
$ yarn
$ yarn start
```

## Build

```
$ yarn build
```
